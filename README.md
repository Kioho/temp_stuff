# The Crazy Candy Store Website

Static multi-page website for The Crazy Candy Store. The site is plain HTML with
inline CSS and JavaScript in each page. Vite is used only for local development,
preview, and production builds.

## Requirements

- Node.js 20, matching `.nvmrc`
- npm 11 or newer

Install dependencies:

```sh
npm install
```

## Commands

```sh
npm run dev
```

Starts the local development server on `127.0.0.1`.

```sh
npm run build
```

Builds all HTML entry points into `dist/`.

```sh
npm run preview
```

Serves the production build locally after `npm run build`.

```sh
npm run check
```

Runs HTMLHint against the five source pages.

```sh
npm run format:check
npm run format
```

Checks or applies Prettier formatting to the HTML files. Formatting the current
pages creates a large mechanical diff, so apply it intentionally.

## Branch Workflow

- Use `Development` as the active working and integration branch.
- Create feature branches from `Development`.
- Open pull requests against `Development`.
- Update `main` by merging `Development` when the live/default GitHub branch
  should be refreshed.

## Project Structure

- `index.html` - Homepage
- `sortiment.html` - Product and price page
- `event.html` - Pre-opening event page
- `om-oss.html` - About page
- `hitta-oss.html` - Location and opening-hours page
- `images/` - Source media used by the pages
- `vite.config.js` - Multi-page Vite build configuration
- `PROJECT_PLAN.md` - Planning document for future changes and launch-readiness
  work
- `AGENTS.md` - Contributor workflow notes for coding agents

## Current Site Behavior

- All five pages use `images/background3.jpeg` as a fixed full-page background
  through `body::before`, darkened with `filter: brightness(.5)`.
- Page content uses semi-transparent surface colors so text remains fully
  opaque while the shared background image shows through.
- The top navigation bar is opaque on every page.
- Footers are intentionally simplified on every page: logo plus the
  store/location copyright line, with no footer navigation menu.
- `event.html` shows `images/1.jpeg` at normal brightness in the event detail
  image area.
- `hitta-oss.html` shows contact info and `Öppettider` before the feature cards,
  uses an opaque `25 Meter Bort` callout, and no longer includes `images/6.jpeg`.
- `om-oss.html` starts with the About intro content only; the former
  `Kom in & häng med oss` intro panel has been removed.

## Notes

- `dist/` is generated by Vite and ignored by Git.
- `files.zip` is currently in the repository root. Do not deploy directly from
  the root unless this file is meant to be public.
- The current pages intentionally share duplicated inline CSS and JavaScript.
  Extracting shared assets is a useful future cleanup, but it should be done as a
  focused refactor.
- Known review items and future ideas are tracked in `PROJECT_PLAN.md`.
